Анимация

Трансформация - процесс изменения или искажения элемента на странице. К ней относятся смещение, искажение, вращение...
rotate() - поворот на определённое кол-во градусов вокруг своей оси
rotateX/Y/Z() - поворот на определённое кол-во градусов вокруг заданной оси
translate[X/Y/Z]() - смещениеэлемента влево, вправо, вверх или вниз (действует также как относительное позиционирование), если указана ось, смещает по ней
scale[X/Y/Z]() - сжимает или растягивает элемент. Если указана ось, то по ней (если значение от 0 до 1, то уменьшает вдоль оси Х
Наклон skew[X/Y]() - смещает верхний край элемента в одну сторону, а нижний в противоположную при положительном значении. Если значение skewX() положительное, наклоняет верхний край влево 
и наоборот. Если значение skewY() положительное, наклоняет левый край вверх и наоборот. Если указана ось, наклоняет по ней
Значение matrix(), для него нет трансформации, matrix(scaleX, skewY, skewX, scaleY, translateX, translateY), например, matrix(1.5, 0.05, 0.5, 1.2, 2.3, 0.01)
Перспектива perspective() задаёт глубину просмотра, т.е. расстояние по оси Z от плоскости экрана до точки сходимости линий. Никаким образом не влияет при поворотах по оси Z
Используется для придания эффекта перспективы для дочерних элементов внутри родительского. perspective-origin изменяет расположение точек сходимости линий. Для координаты Х доступны
следующие значения: по умолчанию 50%, ключевое слово left/right/center эквавалентно 0/100/50%, для Y ключевое слово top/center/bottom - 0/50/100%
Свойства для 3D-трансформации transform-style, backspace-visibility
Переходы позволяют анимировать исходное значение свойства на новое значение с течением времени. Для управления применяются свойства transition и его значения. Краткая форма: transition:
background-color(свойство) 0.3s(продолжительность) linear(функция времени) 0/5s(задержка);
transition-property/duration/timing-function/delay - в каких свойствах должен быть переход/продолжительность перехода/функция времени/задержка
Анимация - процесс перехода от одного набора свойств к другому
@keyframes AnimName {
from {
свойства }
to {
свойства }
}
Свойство animation - для добавления анимации к элементу в сокращённой записи
animation-name -для добавления анимации по имени
animation-timig-function - задаёт плавность воспроизведения между каждой парой ключевых кадров
animation-fill-mode определяет состояние анимации, когда она не воспроизводится
