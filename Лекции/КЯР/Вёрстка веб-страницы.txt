Вёрстка веб-страницы на CSS
    Системы модульной вёрстки
	1) Bootstrap
	    Включает в себя модульную сетку, правила форматирования кнопок, таблиц, предупреждений, значков и ярлыков. Предназначена для создания контента или информации на сайте, который
		занимает всю ширину контейнера. Также содержит компоненты JavaScript
	2) Skeleton
	    Адаптивная система модульной вёрстки, которая позволяет форматировать кнопки, веб-формы и таблицы.
	3) Foundation
	    Адаптивный CSS-что-то-там. Подходит для создания веб-приложений и интерактивных сайтов. Содержит в себе JS-код для создания раскрывающихся списков, графических элементов,
		подсказок и модальных диал(?ог?)овых окон.
	4) Pure
	    Система, позволяющая создавать стили, с помощью которых можно оформлять не только сетки, но и кнопки, таблицы, меню и формы.
	5) Simple Grid
	    Система, которая обеспечивает создание каркаса модульной сетки. Является адаптивной.
Медиазапросы - правила, которые позволяют назначить стили в зависимости от размера окна браузера
    Подключаются 1) атрибут media тега link 2) правило @madia 3) правило @import
	@media {
	    селектор {
		свойства, значения } }
	<style> @import url(styles.css) (max-width:400px);
	    .class {font-size:2px;} </style>
	<link href="styles.css" rel="stylesheet" media="(max-width:400px)">
