Типы шрифтов Антиквенные (с засечками) шрифты идеальны для длинных фрагментов текста, т.к. засечки делают текст более удобочитаемым Рубленые шрифты (без засечек) используются для заголовков благодаря рубленому виду Моноширинный шрифт используется для отображения машинного кода, т.к. каждая буква имеет одинаковую ширину Декоративные шрифты - особые шрифты, которые применяются для визуального выделения заголовков, текстовых эффектов и никогда для основного текста Свойства форматирования списков list-style-type(изменение маркера)/image(использование изображения вместо маркера)/position(позиционирование маркера внутри или снаружи текста списка) Свойства форматирования таблиц padding отступ до границы ячейки text-align горизонтальное выравнивание текста vertiacl-align вертикальное выравнивание текста border-spacing расстояние между ячейками border-collapse удаление интервалов и удвоения границ между ячейками border-radius скругление углов у ячеек empty-cells скрывает пустые ячейки Свойства форматирования форм псевдоклассы focus, checked, enabled... display для добавления элементам label свойств блочных элементов ::valid ::invalid для определения валидности информации, например, для электронной почты width vertical-align text-align margin font-size Свойства форматирования фона background-image/repeat(повторение изображения, если его размер меньше размера элемента)/size(масштаюирует изображение согласно заданным значениям)/position (устанавливает расположение изображения, еси задано background-repeat: no-repeat)/attachment(определяет положение фонового изображения относительно полосы прокрутки (зафиксировано или нет))/origin(определяет размер области, которая будет использоваться фоновым изображением)/clip(как цвет фона или фоновое изображение будут выводиться под границами блока)/blend-mode(устанавливает режим смешивания накладываемых фоновых изображений) гифки - могут включать в себя анимацию, обеспечивают хорошее сжатие, позволяют установить один из цветов прозрачным Корневной относителный путь - л=для доступа к изображениям, хранящимся на своём собственном сайте Относительный от докумета используется без привлечения веб-сервера
