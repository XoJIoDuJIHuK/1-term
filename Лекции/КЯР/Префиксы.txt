Вёрстка веб-страницы на CSS
	Фиксированная вёрстка - когда сайт имеет строго определённыю ширину и высоту всех блоков и страницы и не зависит от размеров экрана
	Резиновая вёрстка - размеры всех элементов задаются не в пикселях, а в процентах
	Адаптивная вёрстка - фактически совокупность резиновой вёрстки и медиазапросов (для определённых размеров экрана устанавливается, как должны выглядеть какие элементы, какие элементы убрать или не убрать)
	Flex-вёрстка - элементы могут растягиваться и сжиматься для заполнения свободного простарнства
		Свойства flex-контейнера:
			display: flex преобразует элемент во flex-контейнер
			flex-flow задаёт направление, в котором элементы будут отображаться, а также их перенос на следующую строку
			flex-direction нарпавление отображения элементов
			flex-wrap как переносятся элементы, когда в родительском нет места
			flex-flow (значения: row отображает элементы последовательно друг за другом, column друг над другом в виде блоков, row(column)-reverse то же самое, но в обратном порядке, 
				nowrap в одну строку независимо от того, насколько узкий браузер, wrap позволяет переноситься, wrap-reverse позволяет переносить элементы в обратном порядке)
			justify-content определяет способ выравнивания flex-элемнтов по ширине (flex-start(end) по левому(правому) краю, center по центру, space-between равномерно распределяет
				свободное пространство между элементами, space-around то же самое, но добавляет также пространство вокруг крайних элементов)
			align-items как flex-элементы различной высоты будут выровнены по высоте строки (flex-start(end), center, baseline выравнивание всех элементов по базовой линии перовго
				элемента, stretch растягивает все элементы под одну высоту
			align-content как браузер будет размещать элементы, занимающие несколько строк
			order определяет порядок следования элементов (как z-index)
			align-self определяет выравнивание по высоте строки отдельных элементов
			flex опредеялет способность гибкого элемента растягиваться/сжиматься
			flex-grow определяет относительную ширину flex-элемента
			flex-shrink определяет степень сжатия элемента
			flex-basis определяет базовую ширину элемента
	Grid-вёрстка определяет разметку с помощью разделения на колонки и строки для последующего добавления элементов
		display-grid преобразует элемент в grid-контейнер
		grid-template-columns(rows) устанавливает размер каждой колонки(строки) ((repeat (5, 25px) - будет 5 колонок по 25 пикселей)
		grid-gap устанавливает промежуток между ячейками сетки
		grid-auto-columns(rows) автоматически определяет размер каждой колонки(строки)
		grid-auto-flow автоматически расставляет элементы
		grid-template-areas устанавлвает сетку в зависимости от имени каждой ячейки
		grid-area устанавливает имя для элемента
		grid-column(row) устанавливает местоположение элемента по номеру колонки(строки)
		align-self, justify-self - выравнивание элемента по вертикали и горизонтали соответственно
